# LMS Frontend

### Setup instruction

1. Clone the project

```
    git clone https://github.com/ChinmayKaitade/LMS-Frontend-React-iNeuron.git
```

2. Move into the directory

```
    cd LMS-Frontend-React-iNeuron
```

3. install dependencies

```
    npm i
```

4. run the server

```
    npm run dev
```
